## 📌 Sobre o projeto

O **Bikcraft** é um projeto fictício de um site para uma empresa de bicicletas elétricas, desenvolvido durante os cursos da Origamid com foco em prática de desenvolvimento front-end e design de interfaces.

A proposta do projeto é simular um site real de mercado, aplicando conceitos modernos de estruturação, estilização e responsividade.
